# cool_prompts

This repo is consolidated into a single product directory:

- Start here: `product/README.md`
- Canonical prompts: `product/prompts/`
- Book index: `product/book/BOOK.md`

Common commands:

- Rebuild the book: `python product/book/_build_book.py`
- Or via the product entrypoint: `python product/product.py build-book`
